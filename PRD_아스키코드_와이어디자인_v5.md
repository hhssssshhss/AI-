# CareerFolio 와이어프레임 (ASCII) — v5

- 기준 문서: PRD_커리어폴리오_v5.md
- v5 핵심 변경: ① AI(Google Gemini API) 실제 연동 ② 파일 스토리지(사용자 Google Drive) 실제 연동
  ③ 로그인은 여전히 완전 목업 ④ 서버 DB·또래 원본 데이터는 여전히 목업(세션 유지)
- 표기: `[ ]` 버튼/토글, `[[ ]]` 주요 CTA, `----` 구분선, `<>` 아이콘 자리
- 화면 하단 안내 문구 범례
  - `<mock>` = 이번 데모에서도 실제로 동작하지 않는 부분 (로그인, DB 영속 저장, 또래 원본 데이터 등)
  - `<real>` = v5부터 실제로 동작하는 부분 (Google OAuth/Drive, Gemini API 호출)

