|  ------------------------------------------------------------------ |

|  종합 갭 스코어                    또래 그룹 대비 강점/약점 (레이더)  |

|  |                  |             |         정량성과            |    |

|  · 자기소개는 평균 3문장 이내로 간결하게 구성                         |

|  ------------------------------------------------------------------ |
